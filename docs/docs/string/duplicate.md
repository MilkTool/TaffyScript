# string.duplicate

[\[global\]]({{site.baseurl}}/docs/).[string]({{site.baseurl}}/docs/string/).[duplicate]({{site.baseurl}}/docs/string/duplicate/)

Returns a string with this string repeated the specified number of times.

```cs
string.duplicate(count)
```
