# Array.set

[\[global\]]({{site.baseurl}}/docs/).[Array]({{site.baseurl}}/docs/Array/).[set]({{site.baseurl}}/docs/Array/set/)

Sets an index in the array to the specified value.

```cs
Array.set(index, value)
```
