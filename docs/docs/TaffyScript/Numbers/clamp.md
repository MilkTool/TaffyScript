# clamp

[\[global\]]({{site.baseurl}}/docs/).[TaffyScript]({{site.baseurl}}/docs/TaffyScript/).[Numbers]({{site.baseurl}}/docs/TaffyScript/Numbers/).[clamp]({{site.baseurl}}/docs/TaffyScript/Numbers/clamp/)

```cs
clamp(value, min, max)
```
