# string_insert

[\[global\]]({{site.baseurl}}/docs/).[TaffyScript]({{site.baseurl}}/docs/TaffyScript/).[Strings]({{site.baseurl}}/docs/TaffyScript/Strings/).[string_insert]({{site.baseurl}}/docs/TaffyScript/Strings/string_insert/)
