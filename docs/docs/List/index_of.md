# List.index_of

[\[global\]]({{site.baseurl}}/docs/).[List]({{site.baseurl}}/docs/List/).[index_of]({{site.baseurl}}/docs/List/index_of/)

_Finds the index of the first occurrence of the value in the list. Returns -1 if the value isn't found._
