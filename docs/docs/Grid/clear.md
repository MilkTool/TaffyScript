# Grid.clear

[\[global\]]({{site.baseurl}}/docs/).[Grid]({{site.baseurl}}/docs/Grid/).[clear]({{site.baseurl}}/docs/Grid/clear/)

Sets all cells in the grid to the specified value.

```cs
Grid.clear(default_value)
```
