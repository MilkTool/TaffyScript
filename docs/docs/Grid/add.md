# Grid.add

[\[global\]]({{site.baseurl}}/docs/).[Grid]({{site.baseurl}}/docs/Grid/).[add]({{site.baseurl}}/docs/Grid/add/)

Adds a given value to the value at the specified position.

```cs
Grid.add(x, y, value)
```
