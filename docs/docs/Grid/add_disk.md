# Grid.add_disk

[\[global\]]({{site.baseurl}}/docs/).[Grid]({{site.baseurl}}/docs/Grid/).[add_disk]({{site.baseurl}}/docs/Grid/add_disk/)

Adds a given value to all values inside of a disk.

```cs
Grid.add_disk(x_middle, y_middle, radius, value)
```
