# Grid.value_exists_in_disk

[\[global\]]({{site.baseurl}}/docs/).[Grid]({{site.baseurl}}/docs/Grid/).[value_exists_in_disk]({{site.baseurl}}/docs/Grid/value_exists_in_disk/)

