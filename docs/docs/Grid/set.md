# Grid.set

[\[global\]]({{site.baseurl}}/docs/).[Grid]({{site.baseurl}}/docs/Grid/).[set]({{site.baseurl}}/docs/Grid/set/)

Sets a position within the grid to a specified value.

```cs
Grid.set(x, y, value)
```
