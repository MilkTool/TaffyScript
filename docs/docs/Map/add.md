# Map.add

[\[global\]]({{site.baseurl}}/docs/).[Map]({{site.baseurl}}/docs/Map/).[add]({{site.baseurl}}/docs/Map/add/)

_Attempts to add an item to the map. Returns true if the item was added, false otherwise._

```cs

```
