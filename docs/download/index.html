Installation

Currently the only type of installation is a standalone exe, so just unpack the zip file wherever you'd like the compiler to be installed and it will be ready to use.

Why are all of the releases for Windows? Right now, TaffyScript is windows only. Work is being done to expand it to MacOS, Linux, iOS, and Android, but it isn't ready for the stamp of approval yet. It should be possible to run tsc using Mono, as well as the resulting programs if it's truly needed right now.
